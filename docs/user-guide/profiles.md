# Profiles

Profiles let you save different mod combinations and switch between them.

![Export Setup](https://i.ibb.co/Nd1RpWTM/image.png)

## Examples

- Vanilla+: few QoL mods
- Heavy Mods: gameplay overhaul
- Testing: temporary setups
